# NotionNext（フォーク版）

このリポジトリは、**NotionNext の日本語化**と、**自分のサイト向けのテーマ改変**を目的として fork したバージョンです。  
Next.js + Notion API を使用した、Vercel などに簡単にデプロイできる静的ブログシステムで、Notion を使うすべてのクリエイターのために設計されています。

---

## 🔧 このフォーク版で行っている主な作業

- NotionNext 全体の日本語訳（README / ドキュメント / UI）を進行中  
- 自分のサイト用にテーマを改変  
  - フッター文言の調整  
  - メニューやナビゲーションの日本語最適化  
  - UI/UX の微修正  
  - 必要に応じて独自コンポーネントを追加
